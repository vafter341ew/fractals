How It Works

Transformations: Each transformation is defined by an affine matrix operation:

[x'] = [a b][x] + [e]
[y'] [c d][y] [f]

Hausdorff Dimension Estimation: D ≈ ln(n) / ln(1/r)

where n = number of transformations and r = average contraction ratio
